回答フォーマット
回答は型抜き情報を次のようにJSON形式データで記録します． 
"n" キーに手数を記録します． "ops" キーに型抜き情報を1手目から順に記録します．
1 回の型抜き情報は，抜き型番号，適用X座標，適用Y座標，寄せ方向（上:0，下:1，左:2，右:3）の4要素で構成され，それらの要素を "p", "x", "y", "s" キーに記録します． 
示した手数と，型抜き情報の個数が異なる回答は無効とします． そのほか，フォーマットに合致しない回答は無効とします． 

例
{ "n":3, "ops":[ { "p":4, "x":5, "y":6, "s":2 }, { "p":11, "x":3, "y":-12, "s":3 }, { "p":25, "x":6, "y":0, "s":0 } ] }
